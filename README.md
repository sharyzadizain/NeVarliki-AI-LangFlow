# Установка Langflow и загрузка JSON-шаблона (flow)

## Установка Langflow

1. Установка Python 3.10–3.12 (рекомендуется 3.11)

Проверка версии:
```bash
python3 --version
```
2. Установка uv

macOS / Linux:
```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```
```bash
Windows (PowerShell):
powershell -c "irm https://astral.sh/uv/install.ps1 | iex"
```

3. Запуск Langflow
```bash
uvx langflow run
```

## Загрузка JSON-шаблона (flow)


1. Открыть Langflow в браузере
2. Нажать кнопку **"New Flow"**
3. В правом верхнем углу нажать **"Import"**
4. Выбрать  JSON-файл
5. Flow загрузится на канвас
